其他包装类缓存机制

除了Integer之外,其他的包装类也使用了缓存技术;

Long

缓存范围-128~127

Byte

缓存范围-128127 (byte范围:一个byte占8位,所以取值范围是**-2^72^7-1**)

Character

缓存范围0~127 (ascii码范围)

Short

缓存范围-128~127

Boolean

缓存范围 true false 它只设置了两个静态变量用来充当缓存

建议

包装类对比数据是否相同的时候,建议采用重写的equals()方法.
